# Guardianes de la Galaxia - Sitio web

Sitio web sobre la película *Guardianes de la Galaxia* (2014). Lo hice para el
segundo parcial de Programación Web 1.

La idea fue armar el sitio con una onda retro-espacial ochentosa (estilo synthwave),
usando colores neón sobre fondo oscuro y algo de brillo en los títulos.

## Páginas

- **Inicio** (`index.html`) - presentación de la película y ficha técnica.
- **Sobre la película** (`sobre-la-pelicula.html`) - sinopsis, producción y un reproductor con música.
- **Reparto** (`reparto.html`) - actores y personajes.
- **Producción** (`produccion.html`) - estudio y datos del rodaje.
- **Curiosidades y premios** (`curiosidades.html`)
- **Contacto** (`contacto.html`) - formulario de contacto.

## Tecnologías

- HTML5 (etiquetas semánticas: header, nav, main, section, article, aside, footer).
- CSS3 (variables, flexbox, transiciones).
- Fuentes de Google Fonts: Orbitron para los títulos e Inter para el texto.

No usé frameworks ni JavaScript, solo HTML y CSS como pedía la consigna.
